    En el este sitio web podra encontrar todos los ejercicios desarrollados en el parcial 1 de estructura de datos 2024A en la universidad de santander (UDES) Todos los ejercicios fueron desarrollados en java y para su comodidad cada uno consta de un boton que le permite copiar en el portapapeles el codigo del ejercicio para su facil revision y reutilizacion


    De la parte izquierda de la pagina encontrara un boton que dice "menu", alli podra encontrar de momento el parcial 1 de estructura de datos 2024. Del lado derecho tambien tendra el boton "home", que al presionar recargara la pagina para volverlo a traer a la pagina principal que es esta


    Este repositorio ha sido desarrollado utilizando tecnologías modernas, como HTML, CSS, JavaScript y Bootstrap, para ofrecer una interfaz intuitiva y fácil de usar.
